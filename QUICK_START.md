# ⚡ QUICK START - TransferIQ Dashboard

## 🎯 3-STEP DEPLOYMENT

### STEP 1: Test Locally (2 minutes)
```bash
cd /path/to/TransferIQ
pip install -r requirements.txt
streamlit run app.py
```
✅ Opens at http://localhost:8501

### STEP 2: Upload to GitHub (5 minutes)
1. Create repo at https://github.com/new
2. Name it: `transferiq-dashboard`
3. Upload all files via web interface
4. Commit changes

### STEP 3: Deploy to Streamlit (3 minutes)
1. Go to https://share.streamlit.io
2. Click "New app"
3. Select your repo
4. Main file: `app.py`
5. Click "Deploy"

## ✅ Your URL will be:
```
https://YOUR-NAME-transferiq-dashboard.streamlit.app
```

**Pages not showing**
→ Check `pages` folder exists with 3 files

**Deployment fails**
→ Repository must be PUBLIC
